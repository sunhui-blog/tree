# tree

<p>树形控件</p>

特性
<ul>
  <li>n(n≥0)个结点的有限集；</li>
  <li>n=0时称为空树；</li>
  <li>在任意一棵的非空树中：
    <ul>
      <li>有且仅有一个特定的称为根(Root)的结点</li>
      <li>当n>1时，其余结点可分为m(m>0)个互不相交的有限集T...,其中每个集合本身又是一棵树，并且称为根的子树(SubTree)</li>
    </ul>
  </li>
</ul>

存储
<ul>
  <li>从内存角度分析</li>
</ul>

树的遍历
<ul>
  <li>前序</li>
  <li>中序</li>
  <li>后序</li>
</ul>

特别的树
<ul>
  <li>二叉树</li>
  <li>线索二叉树</li>
  <li>赫夫曼树</li>
</ul>

属性
<ul>
  <li>树的深度</li>
</ul>

事件
<ul>
  <li>节点增/删</li>
  <li>节点选择器</li>
  <li>节点禁用</li>
  <li>树节点触发</li>
  <li>复选触发</li>
  <li>展开收缩触发</li>
</ul>

获取数据的两种方式
<ul>
  <li>全部获取</li>
  <li>点节点分批获取</li>
</ul>
